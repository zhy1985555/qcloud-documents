## 配置前须知
3336.cnf 文件保存网关的全部配置信息。
Server 模块配置参数为网关的路由模块配置参数。
Gateway 模块配置参数网关的主进程（mysql-proxy）配置参数。Gateway 包含以下模块参数：
- Log
- Mode
- Security
- Statistics
- Flowcontrol
- XA
- Join
