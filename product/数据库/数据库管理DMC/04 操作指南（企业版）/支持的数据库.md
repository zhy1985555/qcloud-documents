| 数据库类型  | 部署形态和版本                                               |
| ----------- | ------------------------------------------------------------ |
| MySQL       | <li>自建 MySQL 5.5、5.6、5.7、8.0 <br><li>云数据库 MySQL 5.5、5.6、5.7、8.0<br/><li>第三方云厂商数据库 MySQL 5.5、5.6、5.7、8.0 |
| TDSQL MySQL | 云数据库 TDSQL MySQL 5.6、5.7、8.0                           |
